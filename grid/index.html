CSS Grid

Conceptos-Grid

Grid Container:

Es similar al de felxbox, es el contenedor principal, en la imagen de abajo se observa con un borde de color verde

Grid item:

Son los elementos hijos directos del container

Grid Cell:

Son las celdas, son elementos representativos dentro del container(no es lo mismo que un grid-item )

Grid Tracks: (column $ row)

Son las columnas y las filas dentro del container.

Grid Area:

Son area que seleccionamos del contenedor

Grid Line:

Son las lineas divisoras de las celdas.

Grid Dinamico

Estos valores van dentro del repat()

minmax( ) = sirve para declarar un tamaño minimo y maximo a los elemtnos

grid-template-columns: repeat(5, minmax(100px, 500px));
grid-template-rows: repeat(3,minmax(100px, 500px));

min-content = este es un valor que va en repeat, y sirve para establecer el tamaño minimo de acuerdo al contenido del item o elemento

grid-template-columns: repeat(5, min-content);
grid-template-rows: repeat(3,min-content);

max-content = este es un valor que va en repeat, y sirve para establecer el tamaño maximo de acuerdo al contenido del item o elemento

grid-template-columns: repeat(5, max-content);
grid-template-rows: repeat(3, max-content);

Cantidades

auto-fill lo que hace es ajustar las columnas de acuerdo al ancho del contenedor (esta propiedad funciona si minmax(###px,###px) esta declarada)

auto-fit funciona similar a auto-fill, solo que este rellene por completo la fila para cada elemento, este se aprecia mejor cuando hay menos elementos en la fila como 2 o 4
