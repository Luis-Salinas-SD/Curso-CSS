CSS Grid

Conceptos-Grid

Grid Container:

Es similar al de felxbox, es el contenedor principal, en la imagen de abajo se observa con un borde de color verde

Grid item:

Son los elementos hijos directos del container

Grid Cell:

Son las celdas, son elementos representativos dentro del container(no es lo mismo que un grid-item )

Grid Tracks: (column $ row)

Son las columnas y las filas dentro del container.

Grid Area:

Son area que seleccionamos del contenedor

Grid Line:

Son las lineas divisoras de las celdas.


Display Grid / Grid Templates

Displya: grid;

Cuando le colocamos al contenedor display:grid; lo unico que hacemos es que cree filas.

Medidas

grid-template-columns:
